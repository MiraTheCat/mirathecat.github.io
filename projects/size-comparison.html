Controls:


Normal Mode

- Click on the right side of the screen to go right, and the left side of the screen to go left
- You can change which Abomination the comparison starts on in the settings by clicking the gear icon

Custom Mode

- When creating a custom comparison, click on the left/right side of the screen to switch between the abominations

Saving and Loading

- To save a custom comparison, click the save button and copy the save code
- For big comparisons, it may take a few seconds for the save code to generate
- The save code will disappear after 10 seconds or by clicking a button
- Load the custom comparison by pasting the save code after clicking the load button
